Fighting for the working class...
This design for an online discussion board opens with the prompt asking what it means to fight for the working class in 2021. Scrolling down, the visitor can read the submitted entries which highlight issues that American workers are facing. This website aims to provide a space not only for demonstrating a range of problems, but also for proposing solutions: what are the actions we can take now to fight for change? Arriving at the bottom of the webpage, the user is now urged to write and publish their own response.
